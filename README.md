# XML to Excel Converter

A simple cross-platform Flutter application that allows users to upload an XML file, parses its structured data, and converts it into a downloadable Excel (`.xlsx`) file.

## Features
- **File Upload:** Select `.xml` files from your device.
- **Data Parsing:** Automatically extracts headers and rows from flat XML structures.
- **Excel Generation:** Creates and downloads an `.xlsx` file directly in the browser or on device.

## Getting Started

1. Ensure you have Flutter installed.
2. Clone the repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to start the app.
